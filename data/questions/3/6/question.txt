<p style="margin-bottom: 0cm; border: none; padding: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; font-weight: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="2" style="font-size: 10pt"><font color="#222222">You
have an array&nbsp;<font size="3" style="font-size: 12pt"><i>a</i></font>&nbsp;consisting
of&nbsp;<font size="3" style="font-size: 12pt"><i>n</i></font>&nbsp;integers.
Each integer from&nbsp;<font size="3" style="font-size: 12pt">1</font>&nbsp;to&nbsp;<font size="3" style="font-size: 12pt"><i>n</i></font>&nbsp;appears
exactly once in this array.</span></font></font></font></p>
<p style="margin-bottom: 0cm; border: none; padding: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; font-weight: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="2" style="font-size: 10pt"><font color="#222222">For
some indices&nbsp;<font size="3" style="font-size: 12pt"><i>i</i></font>&nbsp;(<font size="3" style="font-size: 12pt">1 ≤ </font><font size="3" style="font-size: 12pt"><i>i</i></font><font size="3" style="font-size: 12pt"> ≤ </font><font size="3" style="font-size: 12pt"><i>n</i></font><font size="3" style="font-size: 12pt"> - 1</font>)
it is possible to swap&nbsp;<font size="3" style="font-size: 12pt"><i>i</i></font>-th
element with&nbsp;<font size="3" style="font-size: 12pt">(</font><font size="3" style="font-size: 12pt"><i>i</i></font><font size="3" style="font-size: 12pt"> + 1)</font>-th,
for other indices it is not possible. You may perform any number of
swapping operations any order. There is no limit on the number of
times you swap&nbsp;<font size="3" style="font-size: 12pt"><i>i</i></font>-th
element with&nbsp;<font size="3" style="font-size: 12pt">(</font><font size="3" style="font-size: 12pt"><i>i</i></font><font size="3" style="font-size: 12pt"> + 1)</font>-th
(if the position is not forbidden).</span></font></font></font></p>
<p style="margin-bottom: 0cm; border: none; padding: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; font-weight: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="2" style="font-size: 10pt"><font color="#222222">Can
you make this array sorted in ascending order performing some
sequence of swapping operations?</span></font></font></font></p>
<p style="margin-bottom: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="3" style="font-size: 12pt"><b><font color="#222222">Input</span></b></font></font></font></p>
<p style="margin-bottom: 0cm; border: none; padding: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; font-weight: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="2" style="font-size: 10pt"><font color="#222222">The
first line contains one integer&nbsp;<font size="3" style="font-size: 12pt"><i>n</i></font>&nbsp;(<font size="3" style="font-size: 12pt">2 ≤ </font><font size="3" style="font-size: 12pt"><i>n</i></font><font size="3" style="font-size: 12pt"> ≤ 200000</font>)
— the number of elements in the array.</span></font></font></font></p>
<p style="margin-bottom: 0cm; border: none; padding: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; font-weight: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="2" style="font-size: 10pt"><font color="#222222">The
second line contains&nbsp;<font size="3" style="font-size: 12pt"><i>n</i></font>&nbsp;integers&nbsp;<font size="3" style="font-size: 12pt"><i>a</i></font><sub><font size="2" style="font-size: 9pt"><i>1</i></font></sub>,&nbsp;<font size="3" style="font-size: 12pt"><i>a</i></font><sub><font size="2" style="font-size: 9pt"><i>2</i></font></sub>,
...,&nbsp;<font size="3" style="font-size: 12pt"><i>a</i></font><sub><font size="2" style="font-size: 9pt"><i>n</i></font></sub>&nbsp;(<font size="3" style="font-size: 12pt">1 ≤ </font><font size="3" style="font-size: 12pt"><i>a</i></font><sub><font size="2" style="font-size: 9pt"><i>i</i></font></sub><font size="3" style="font-size: 12pt"> ≤ 200000</font>)
— the elements of the array. Each integer from&nbsp;<font size="3" style="font-size: 12pt">1</font>&nbsp;to&nbsp;<font size="3" style="font-size: 12pt"><i>n</i></font>&nbsp;appears
exactly once.</span></font></font></font></p>
<p style="margin-bottom: 0cm; border: none; padding: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; font-weight: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="2" style="font-size: 10pt"><font color="#222222">The
third line contains a string of&nbsp;<font size="3" style="font-size: 12pt"><i>n</i></font><font size="3" style="font-size: 12pt"> - 1</font>&nbsp;characters,
each character is either&nbsp;<font size="3" style="font-size: 11pt">0</font>&nbsp;or&nbsp;<font size="3" style="font-size: 11pt">1</font>.
If&nbsp;<font size="3" style="font-size: 12pt"><i>i</i></font>-th
character is&nbsp;<font size="3" style="font-size: 11pt">1</font>,
then you can swap&nbsp;<font size="3" style="font-size: 12pt"><i>i</i></font>-th
element with&nbsp;<font size="3" style="font-size: 12pt">(</font><font size="3" style="font-size: 12pt"><i>i</i></font><font size="3" style="font-size: 12pt"> + 1)</font>-th
any number of times, otherwise it is forbidden to swap&nbsp;<font size="3" style="font-size: 12pt"><i>i</i></font>-th
element with&nbsp;<font size="3" style="font-size: 12pt">(</font><font size="3" style="font-size: 12pt"><i>i</i></font><font size="3" style="font-size: 12pt"> + 1)</font>-th.</span></font></font></font></p>
<p style="margin-bottom: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="3" style="font-size: 12pt"><b><font color="#222222">Output</span></b></font></font></font></p>
<p style="margin-bottom: 0cm; border: none; padding: 0cm; font-variant: normal; letter-spacing: normal; font-style: normal; font-weight: normal; orphans: 2; widows: 2">
<span style="display: inline-block; border: none; padding: 0cm"><font face="Arial, sans-serif"><font size="2" style="font-size: 10pt"><font color="#222222">If
it is possible to sort the array in ascending order using any
sequence of swaps you are allowed to make, print&nbsp;<font size="3" style="font-size: 11pt">YES</font>.
Otherwise, print&nbsp;<font size="3" style="font-size: 11pt">NO</font>.</span></font></font></font></p>
